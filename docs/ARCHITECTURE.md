# 系统架构

## 总体结构

本项目面向轮腿机器人整机系统，预计包含以下子系统：

- 机械结构与轮腿执行机构。
- 电机驱动板与功率电子。
- STM32 电机控制与 FOC 固件。
- 上位机或主控计算平台。
- ROS2 运动控制、状态估计、仿真或可视化软件。
- 标定、分析、自动化测试和实验记录工具链。
