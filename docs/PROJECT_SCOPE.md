# 项目范围

## 项目目标

建立轮腿机器人整机研发仓库，统一管理固件、硬件、机械、ROS2、测试、工具和实验记录。

## 当前纳入范围

- STM32 电机控制与 FOC 固件工程。
- 硬件设计资料、器件库、驱动板资料、BOM 和数据手册。
- 机械 CAD、STL 网格、装配相关资料和机器人描述资料。
- ROS2 工作区骨架。
- 自动化测试目录骨架。
- 标定、分析和辅助工具目录骨架。
- 实机实验记录目录和模板。

## 当前不做事项

- 不修改固件源码。
- 不迁移旧 STM32 工程。
- 不迁移 `mechanical/robot1_description/`。
- 不清理已有编译产物和临时文件。
- 不导入新的机械或硬件大文件。
- 不初始化 Git，不提交 Git。
