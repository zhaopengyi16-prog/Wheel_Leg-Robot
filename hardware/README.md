# 硬件目录

本目录用于保存电子硬件、驱动板、器件库、接线、BOM、数据手册和生产资料。

## 当前结构

- `器件库/`：包含原理图库和封装库。
- `芯片数据手册/`：包含芯片规格书和数据手册。
- `驱动工程/`：包含驱动板工程文件、BOM、PDF、DXF、压缩包和生产相关资料。

## 管理建议

- Altium 设计源文件、BOM 和生产资料应明确版本。
- 大型或二进制设计文件后续应配合 Git LFS 管理。
- 临时预览、历史缓存和 Office 临时文件不应纳入版本管理。
